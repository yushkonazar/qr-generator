# QR Generator

<div align="center">

![Screenshot](screenshot.png)

</div>

A simple and clean web application that generates QR codes from any URL. Enter a link, hit **Generate**, and download your QR code as a PNG image in one click.

## 🔗 Live Demo

👉 [Open on GitHub Pages](https://yushkonazar.github.io/qr-generator/)

## ✨ Features

- Generate a QR code from any URL instantly
- Set a custom filename for the downloaded image
- One-click PNG download
- Smooth animations and transitions
- Responsive design — works on desktop and mobile

## 🛠 Tech Stack

- HTML5
- CSS3 (custom animations, responsive layout)
- JavaScript (ES Modules)
- [QRCode.js](https://github.com/davidshimjs/qrcodejs) — QR code generation library

## 🚀 Getting Started

**Option 1 — GitHub Pages (no setup needed):**

👉 [https://yushkonazar.github.io/qr-generator/](https://yushkonazar.github.io/qr-generator/)

**Option 2 — Run locally:**

1. Clone the repository:
   ```bash
   git clone https://github.com/yushkonazar/qr-generator.git
   cd qr-generator
   ```
2. Open `index.html` in your browser, or use a local server such as [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) for VS Code.

> ⚠️ The project uses ES Modules (`type: "module"`), so opening `index.html` directly via `file://` may cause a CORS error in some browsers. A local server is recommended.

## 📋 How to Use

1. Paste or type a URL into the input field
2. *(Optional)* Enter a custom filename
3. Click **Generate**
4. Click the QR code image to download it as PNG
