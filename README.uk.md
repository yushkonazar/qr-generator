Простий та охайний вебзастосунок для генерації QR-кодів з будь-якого URL. Вставте посилання, натисніть **Generate** — і завантажте QR-код одним кліком у форматі PNG.

## 🔗 Демо

👉 [Відкрити на GitHub Pages](https://yushkonazar.github.io/qr-generator/)

## ✨ Функціональність

- Миттєва генерація QR-коду з будь-якого URL
- Кастомне ім'я для файлу завантаження
- Завантаження у форматі PNG одним кліком
- Плавні анімації та переходи
- Адаптивний дизайн — працює на десктопі та мобільному

## 🛠 Технології

- HTML5
- CSS3 (кастомні анімації, адаптивний макет)
- JavaScript (ES Modules)
- [QRCode.js](https://github.com/davidshimjs/qrcodejs) — бібліотека для генерації QR-кодів

## 🚀 Запуск

**Варіант 1 — GitHub Pages (без налаштувань):**

👉 [https://yushkonazar.github.io/qr-generator/](https://yushkonazar.github.io/qr-generator/)

**Варіант 2 — Локальний запуск:**

1. Клонуй репозиторій:
   ```bash
   git clone https://github.com/yushkonazar/qr-generator.git
   cd qr-generator
   ```
2. Відкрий `index.html` у браузері або запусти локальний сервер (наприклад [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) для VS Code).

> ⚠️ Проект використовує ES Modules (`type: "module"`), тому відкриття `index.html` напряму через `file://` може спричинити CORS-помилку в деяких браузерах. Рекомендується локальний сервер.

## 📋 Як користуватись

1. Вставте або введіть URL у поле вводу
2. *(Необов'язково)* Вкажіть ім'я файлу
3. Натисніть **Generate**
4. Клікніть на QR-код, щоб завантажити його у форматі PNG
